# HDL Cholesterol Prediction

HDL cholesterol prediction from the 2026 ASA Student Data Challenge

## Overview

This project builds regression models to predict HDL cholesterol levels using clinical and demographic features.

The goal is to compare multiple regression models and evaluate predictive performance using appropriate error metrics.

---

## Dataset

The dataset contains patient-level health measurements and demographic information used to predict HDL cholesterol levels.

The full dataset is not included due to GitHub size limits.


---

## Models Used

- Linear Regression  
- Ridge Regression  
- Random Forest Regressor  
- XGBoost Regressor  

---

## Evaluation Metrics

- RMSE (Root Mean Squared Error)
